# werewolf

## 開発状態
現在、C言語によって簡単な機能はVSCode内で入出力できるまでは開発している。<br>
現在(2022/10/13～)、WebアプリとしてデプロイするためにC言語をWebアプリ開発として使った経験のあるPythonに変えていっている。

## 開発内容
「人狼ゲーム」のルールにて、部屋の人数や占い師の占い内容などの情報を入力すると、それらの情報によって得られる役職情報が出力される。
